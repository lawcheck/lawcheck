# Маркетинговое ревью лендинга LawCheck

_Источник: объединённое дизайн+маркетинг ревью прототипа Landing / Pricing / Report, дата ревью — 2026-05-19. Перенесено из knowledge base (`wiki/concepts/lawcheck-landing-audit-2026-05.md`)._

## TL;DR

Проект продаёт «инструмент», а не «защиту от штрафа» — это режет конверсию РФ-ЦА. Дизайн в целом чистый, но фейлит WCAG AA и теряет SMB-доверие из-за «инженерного» микрокопирайта.

ЦА: владельцы интернет-магазинов, клиник, курсов + агентства (тариф Business). Конкурент: vlip.site.

## Три самых надёжных правки (сошлись и дизайнер, и маркетолог)

1. **Микрокопирайт латиницей** (`critical`, `severity`, `where`, `JSON`, `v3.14 · 30 правил` в hero) одновременно а) проваливает доверие SMB и б) выглядит как «техно-вайб», создаёт дистанцию.
2. **Hero-форма перегружена** — поля «Глубина» / «Регион» дизайнерски ломают ритм и маркетингово увеличивают трение.
3. **CTA-иерархия размыта** — `bg-ink` и `bg-brand` спорят на одной странице (Report: «PDF с подписью» vs «Включить мониторинг»).

## Приоритеты дизайна (блокеры доступности и системности)

### P0 — блокеры WCAG / UX

1. Контраст `ink-faint` (2.83:1 при норме 4.5:1) и цветной текст на `*-soft` фонах.
2. `focus-visible:ring` отсутствует везде — блокер WCAG 2.4.7.
3. Мобильная навигация отсутствует (`hidden md:flex` без бургера).
4. Состояния формы (loading / disabled / error / empty) не написаны.

### P1 — системные

5. Дизайн-система в развале: ~24 размера шрифта, 6 радиусов — нужна нормализация токенов.

## A/B-гипотезы (по убыванию ожидаемого impact)

1. **«Суммарный риск штрафа» в Free-отчёте.** CR отчёт → pricing, ожидание +30–50% — самый сильный триггер upsell.
2. **H1 риск-фрейм vs «соответствие 152-ФЗ».** CR hero → submit, ожидание +20–35%.
3. **Анонимный юрист → имя/фото/регалии.** CR pricing → Pro trial, +25%.
4. **Минимальная hero-форма (только URL).** CR submit, +10–15%.
5. **Секция «Сколько стоят нарушения» вместо «30/6/153».** scroll-depth + CR footer-CTA, +15%.

## Открытые вопросы

- vlip.site — реальная структура воронки и pricing-страница не проверены. Нужен отдельный конкурентный аудит.
- «1200+ сайтов проверено» — есть ли реальная цифра, можно ли публиковать.
- Юрист-партнёр — имя/фото/регалии: есть ли согласие на публикацию.
