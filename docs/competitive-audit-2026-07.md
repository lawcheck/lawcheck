# Конкурентный аудит: vlip.site и поле «проверка сайта на 152-ФЗ»

_Дата аудита: 2026-07-08. Метод: разбор публичных страниц (лендинги, оферты,
тарифы) + веб-поиск. Цены и фичи в этой нише меняются быстро — при использовании
позже осени 2026 перепроверить. Закрывает открытый вопрос из
`docs/marketing-review.md` (2026-05-19)._

## TL;DR

1. **vlip.site — не коммерческий конкурент.** Сервис полностью бесплатен
   (ИП Уфимцев С.О., Екатеринбург), 20 проверок за 2 минуты, монетизации нет:
   оферта лишь резервирует право «ввести платные функции» (упомянута ЮКасса),
   API — форма вейтлиста без цен. Его угроза — не отъём выручки, а
   **обесценивание категории**: приучает рынок, что проверка должна быть бесплатной.
2. **Реальный коммерческий соперник — help152.ru**: разовые отчёты
   1 490 / 2 990 / 4 900 ₽, генератор 57 документов за 2 500 ₽, заявляет
   «70+ проверок». Его free-уровень — скоринг 0–100 **без деталей нарушений**.
3. **Никто на рынке не продаёт подписку с мониторингом.** Все конкуренты —
   разовые отчёты или бесплатные чекеры. LawCheck единственный с recurring
   (Pro 990 ₽/мес: мониторинг + diff в Telegram) и живым юристом-партнёром.
   Это главный дифференциатор — и его надо проговаривать на каждой странице.

## vlip.site — глубокий разбор

**Позиционирование.** H1: «Проверьте сайт на соответствие законодательству РФ».
Категория та же, что у нас: автоматическая проверка по 152-ФЗ / ЗОЗПП / 436-ФЗ +
347-ФЗ (у LawCheck вместо него 38-ФЗ «О рекламе» и ЕГРЮЛ). 20 параметров против
наших 30.

**Воронка.** Одна страница: URL → чекбоксы согласий → «Проверить» → бесплатный
отчёт. Дальше только два крючка: согласие на email-рассылку (сбор базы) и
страница `/api-access` — форма заявки «мы свяжемся, как только появится такая
возможность» (имя, email, Telegram, число проверок в месяц — прощупывают спрос
агентств). Нет: тарифов, кабинета, PDF, документов, юруслуг, соцдоказательств
(ни цифр, ни отзывов, ни логотипов).

**Монетизация.** По оферте функционал «предоставляется безвозмездно»; исполнитель
«вправе в будущем ввести платные функции или тарифы», оплата планируется через
ЮКассу. Перевод: монетизация задумана, но не запущена.

**Сильное:** бесплатность без ограничений (лучший магнит по запросам со словом
«бесплатно»), скорость, охват нескольких законов.
**Слабое:** нет модели дохода → нет ресурса углублять продукт; нет юридической
интерпретации; ноль доверия на странице; один ИП без команды.

**Gating-стратегии.** help152 прячет детали (оценка есть — нарушений не видно),
saitscan прячет описания и рекомендации, quickaudit ограничивает глубину обхода.
LawCheck отдаёт находки с цитатами бесплатно и продаёт следующий шаг (PDF,
мониторинг, юрист). Наш free — самый щедрый на рынке: лучший лид-магнит, но
он работает только при сильном upsell-триггере (суммарный риск штрафа в отчёте —
уже реализовано).

## Возможности

1. **«Единственные, кто следит за сайтом дальше».** Никто из поля не мониторит
   сайт после проверки. Формула для лендинга/pricing: «Проверить сайт сегодня
   может любой сервис. Заметить, что завтра он перестал соответствовать, — только
   LawCheck».
2. **Страница-сравнение** «LawCheck vs бесплатные проверки 152-ФЗ» — честная
   таблица (см. выше) + SEO-статья «Что не покажет бесплатная проверка на
   152-ФЗ». Перехват запросов «проверка 152-ФЗ бесплатно», по которым сейчас
   собирают трафик vlip и 1ps.
3. **Разовый платный отчёт как средняя ступень.** Рынок приучен к разовым
   300–4 900 ₽, у нас между free и подпиской 990 ₽/мес ступени нет. Гипотеза:
   разовый PDF-отчёт ~1 990 ₽ (у help152 «Стандарт» — 2 990 ₽). Требует решения
   владельца — может каннибализировать Pro.
4. **Генератор/пакет документов** — help152 продаёт 57 документов за 2 500 ₽
   отдельным продуктом; у нас шаблоны заперты в Pro. Смежная выручка.
5. **Гонка цифр**: help152 заявляет «70+ проверок» против наших «30». Не
   ввязываться в счёт — наша защита: «не галочки, а конкретные цитаты со страниц
   и ссылки на нормы».

## Угрозы

- **vlip включит монетизацию** (ЮКасса уже в оферте): из бесплатного станет
  дешёвым платным с готовой базой email-подписчиков.
- **Коммодитизация проверки**: бесплатные чекеры двигают цену «просто проверки»
  к нулю. Ценность смещается в интерпретацию, мониторинг и юриста — туда, где
  LawCheck уже стоит; важно не позиционироваться как «ещё один чекер».
- **help152 масштабируется**: «скидки в 2 раза» на всех тарифах — активно
  экспериментируют с ценой; генератор документов закрывает потребность, которой
  у нас нет в открытой продаже.
